## 🛠️ Technical Indexing

### 1. Sitemap (`/sitemap.xml`)
The sitemap is dynamically generated via `app/sitemap.ts` to ensure search engines prioritize the main application route with a weight of `1.0`.

### 2. Robots (`/robots.txt`)
Configured to allow all crawlers (`User-Agent: *`) and explicitly point to the canonical sitemap.

### 3. PWA Manifest (`/manifest.webmanifest`)
The manifest includes `standalone` display mode and high-resolution icons (1024x1024) to ensure the app is recognized as a high-quality Progressive Web App.
