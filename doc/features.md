# Premium Features

The Local Markdown Viewer is packed with high-end features tailored for the ultimate writing experience.

## 💧 Liquid Glassmorphism UI
A state-of-the-art design language that provides an immersive user experience:
- **Floating Toolbar**: Context-aware floating glass navigation.
- **Backdrop Blur**: High-performance blur layers for deep visual hierarchy.
- **Theme Awareness**: Deep liquid gradients that morph seamlessly between Light and Dark modes.
- **Glassmorphic Panes**: Semi-transparent editor and preview containers.

## 🚀 Native File Interactivity
- **File System Access API**: Open and Save files directly to your machine with native dialogs.
- **Drag & Drop**: Instant file loading by dropping any `.md` file onto the browser window.
- **Autosave**: A persistent drafting system in `localStorage` that protects your work even on refresh.
- **Self-Healing Guide**: The app automatically repairs the "Welcome Guide" if local storage versions become corrupted or outdated.

## 🧪 Advanced Markdown Support
- **Full KaTeX Support**: High-fidelity mathematical typesetting for scientific writing.
- **Mermaid Diagrams**: Live rendering of flowcharts, sequence diagrams, and more.
- **GitHub Alerts**: Dedicated support for modern GitHub-style blocks (`[!TIP]`, `[!IMPORTANT]`, `[!WARNING]`).
- **Responsive Split View**: Fluid resizing of the editor and preview panes.

## 📦 PWA & Performance
- **Installability**: Install to your desktop or mobile as a standalone application.
- **Offline Support**: Robust service workers ensure the app is always available.
- **Zero Privacy Risk**: Entirely client-side; no servers involved in your writing process.
